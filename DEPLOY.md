# Free Deployment Plan

Best free path for this exact static site: `GitHub Pages`.

## Recommended target

Create a public repo named:

`Naksh7Gupta.github.io`

That special repo name makes GitHub Pages serve the site directly at:

`https://naksh7gupta.github.io/`

This is the cleanest free option for a searchable personal portfolio.

## Files to upload

Upload everything from this folder:

- `index.html`
- `styles.css`
- `script.js`
- `robots.txt`
- `sitemap.xml`
- `site.webmanifest`
- `favicon.svg`

## GitHub Pages steps

1. Create a new public repo named `Naksh7Gupta.github.io`
2. Upload all files from this folder to the repo root
3. Open `Settings`
4. Open `Pages`
5. Under `Build and deployment`, choose `Deploy from a branch`
6. Select branch `main` and folder `/ (root)`
7. Save
8. Wait a minute and open `https://naksh7gupta.github.io/`

## For Google search

This setup helps, but no one can guarantee instant ranking for `Naksh Gupta`.

To improve the chances:

1. Keep `Naksh Gupta` in the page title and visible page copy
2. Link this site from your GitHub profile README
3. Link this site from your YouTube channel bio/about
4. After deploy, add it to Google Search Console
5. Submit `https://naksh7gupta.github.io/sitemap.xml` inside Search Console
6. Share the site publicly so Google sees backlinks and visits
